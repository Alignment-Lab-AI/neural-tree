# iter

Iterate over the dataset.



## Parameters

- **queries**

    List of queries paired with documents.

- **documents**

    List of documents paired with queries.

- **batch_size** – defaults to `512`

    Size of the batch.

- **epochs** (*int*) – defaults to `1`

    Number of epochs.

- **shuffle** – defaults to `True`

- **tqdm_bar** – defaults to `True`
